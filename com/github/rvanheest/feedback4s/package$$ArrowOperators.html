ArrowOperators

Related Doc: package feedback4s

implicit class ArrowOperators[I, O] extends AnyRef

Adds operators for composition of Components in various ways.

Examples:

import nl.rvanheest.feedback4s.Component

val x = new Component[Int, String](in => in.map(_.toChar).scan("")(_ + _).drop(1))
val y = Component.create[String, Int](_.length)
val z = Component.identity[Int]

val a1: Component[Int, Int] = x >>> y
val a2: Component[Int, Int] = x.concat(y)
val b: Component[(Int, Long), (String, Long)] = x.first[Long]
val c: Component[(Long, Int), (Long, String)] = x.second[Long]
val d1: Component[(Int, String), (String, Int)] = x *** y
val d2: Component[(Int, String), (String, Int)] = x.split(y)
val e1: Component[Int, (String, Int)] = x &&& z
val e2: Component[Int, (String, Int)] = x.fanout(z)
val f: Component[Int, Int] = x.combine(z)((s, i) => s.length + i)

  3. def &&&[X](other: Component[I, X]): Component[I, (O, X)]

    Vertically composes src with other such that the input of the wrapping Component is emitted to both src and other.

    Vertically composes src with other such that the input of the wrapping Component is emitted to both src and other. For this to work, both Components must have the same input type. The outputs of the components are combined in a tuple.

    This is the symbolic equivalent of the fanout operator.

    X

    the output type of other

    other

    the other Component in this composition

    returns

    a Component that wraps around the composition of src and other and sends its input to both src and other
